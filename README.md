# 脚本用于Mac版本JY在音频上自动打标记

把脚本和搜索图片放在同一个目录中运行“python3 capcut_auto.py”即可

如果搜索图片和自己的JY搜索不一致，自行替换即可

说明：（该脚本目前只在Mac环境中使用，Windows未进行验证）
1、全屏JY；
2、点击字幕可以显示字幕搜索框；
3、音频放在上面，多段需要合并成一段；
4、我使用过程中遇到的就上面的问题，其他问题联系单独联系我吧（diqiuhao：hlc12336_）。

-----
安装说明
一、脚本清单：
capcut_auto.py （核心脚本）
requirements.txt （依赖列表）
search_box.png （搜索框截图，注意最好去新电脑上重新截一张替换掉，因为不同电脑的屏幕分辨率不同，截图可能会长得不一样）
install.sh （一键安装脚本）

二、操作步骤：
在电脑上打开终端（Terminal），进入到解压后的文件夹里
运行下面这个命令来自动安装环境：
bash
./install.sh
(或者运行 bash install.sh)
最关键的一步：正如安装脚本最后会提示你的，因为脚本需要控制键盘鼠标和识别屏幕颜色，你在新电脑上首次运行前，必须去 Mac 的 系统设置 -> 隐私与安全性 里面：
在【辅助功能】里把终端（Terminal）勾选上。
在【屏幕录制】里也把终端（Terminal）勾选上。
安装环境搞定且给了权限之后，
执行 python3 capcut_auto.py 就可以运行了！
